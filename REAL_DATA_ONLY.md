## 📋 使用流程

### 1. 部署合约
```bash
cd packages/contracts
npm run deploy:sepolia
```

### 2. 设置合约地址
- 在前端界面输入部署的合约地址
- 点击"设置"按钮

### 3. 创建提案
- 点击"+ 创建提案"按钮
- 填写提案标题和描述
- 选择投票时长
- 点击"创建提案"

### 4. 投票
- 在提案卡片中选择"支持"或"反对"
- 点击投票按钮
- 确认交易

### 5. 查看结果
- 投票结束后，结果会自动显示
- 显示每个选项的投票数量和百分比
