## 🚀 快速开始

### 1. 部署合约

```bash
cd packages/contracts
npm run deploy:sepolia
```

部署成功后，复制合约地址。

### 2. 启动前端

```bash
cd packages/frontend
npm run dev
```

### 3. 使用投票平台

1. **连接钱包** - 点击"连接钱包"按钮
2. **设置合约地址** - 输入部署的投票合约地址
3. **创建提案** - 点击"+ 创建提案"，填写标题和描述
4. **投票** - 选择"支持"或"反对"，点击投票
5. **查看结果** - 投票结束后自动显示结果

## 🎯 操作对比

### 投票平台（新）✅
- **创建提案**: 填写标题+描述 → 点击创建（2步）
- **投票**: 选择选项 → 点击投票（2步）
- **查看结果**: 自动显示（0步）
- **总步骤**: 5步，单人即可完成

### 拍卖系统（旧）❌
- **创建拍卖**: 填写3个字段 → 创建（3步）
- **出价**: 输入金额 → 加密 → 提交（3步）
- **结束拍卖**: 结束 → 确定获胜者 → 结算（3步）
- **总步骤**: 8步，需要多人配合

## 🎨 界面说明

### 主页面
- 顶部有模式切换按钮：🗳️ 投票平台（推荐）和 🔐 拍卖系统
- 默认使用投票平台模式

### 创建提案
- 点击"+ 创建提案"按钮
- 填写提案标题和描述
- 选择投票时长（1小时/1天/1周）
- 可选：启用加权投票
- 点击"✅ 创建提案"

### 投票
- 在提案卡片中，选择"✅ 支持"或"❌ 反对"
- 点击后自动加密并提交
- 显示"✅ 您已投票，感谢参与！"

### 查看结果
- 投票结束后，自动显示结果
- 显示每个选项的投票数量和百分比
- 可视化图表展示

## 🔧 技术实现

### 智能合约
- **文件**: `packages/contracts/contracts/PrivacyVoting.sol`
- **主要函数**:
  - `createProposal()` - 创建提案
  - `submitVote()` - 提交加密投票
  - `endProposal()` - 结束提案
  - `finalizeProposal()` - 结算提案
  - `getVotesBatch()` - 批量获取投票

### 前端组件
- **文件**: `packages/frontend/src/components/VotingPlatform.tsx`
- **主要功能**:
  - 提案列表显示
  - 创建提案表单
  - 投票界面
  - 结果可视化

### 合约工具
- **文件**: `packages/frontend/src/utils/votingContract.ts`
- **主要函数**:
  - `getVotingContract()` - 获取合约实例
  - `VOTING_ABI` - 合约 ABI
