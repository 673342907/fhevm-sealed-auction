# 使用指南

## 快速开始

### 1. 连接钱包

1. 打开应用
2. 点击"连接钱包"按钮
3. 在 MetaMask 中确认连接
4. 确保切换到 Sepolia 测试网

### 2. 设置合约地址

1. 在"合约地址配置"中输入已部署的合约地址
2. 点击"设置"按钮

### 3. 创建拍卖

1. 填写物品名称和描述
2. 设置拍卖时长（秒）
3. 点击"创建拍卖"按钮
4. 在 MetaMask 中确认交易

### 4. 参与竞标

1. 在拍卖卡片中找到要参与的拍卖
2. 输入出价金额
3. 点击"提交加密出价"
4. 出价将被自动加密后提交到链上

### 5. 结束拍卖

1. 等待拍卖时间结束
2. 点击"结束拍卖"按钮
3. 系统会自动比较所有出价
4. 确定最高出价者

## 功能说明

### 加密出价

- 所有出价在提交前自动加密
- 加密数据存储在区块链上
- 在拍卖结束前，出价信息保持加密状态

### 隐私保护

- 其他参与者无法看到你的出价
- 只有拍卖结束后才会揭示结果
- 使用 EIP-712 签名确保解密安全

### 拍卖状态

- **进行中**: 可以提交出价
- **已结束**: 等待结果揭示
- **已结算**: 拍卖完成，结果已确定

## 注意事项

1. **Gas 费用**: 加密操作需要较高的 Gas 费用，确保账户有足够的 ETH
2. **网络**: 目前仅支持 Sepolia 测试网
3. **出价限制**: 每个地址只能对同一拍卖出价一次
4. **时间限制**: 必须在拍卖结束时间前提交出价

## 常见问题

### Q: 为什么我的出价没有显示？

A: 出价是加密存储的，在拍卖结束前不会显示具体金额。你可以在交易记录中查看提交状态。

### Q: 如何查看我的出价？

A: 在拍卖结束后，系统会揭示所有出价。你可以使用解密功能查看自己的出价。

### Q: 如果拍卖时间到了怎么办？

A: 任何人都可以调用"结束拍卖"函数。系统会自动处理后续流程。

### Q: 如何获得 Sepolia ETH？

A: 可以使用 Sepolia 水龙头获取测试 ETH：
- https://sepoliafaucet.com/
- https://faucet.quicknode.com/ethereum/sepolia
